# THE language - minimal skeleton

This repository contains a tiny interpreter skeleton for a toy language `THE`.

Run the example:

```bash
python run.py
```

Files created:

- main.py
- run.py
- lexer/token.py, lexer/tokenizer.py
- parser/parser.py, parser/ast_nodes.py
- runtime/interpreter.py, runtime/environment.py
- utils/errors.py
- examples/hello.the
